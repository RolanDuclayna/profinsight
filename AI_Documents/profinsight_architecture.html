Content Script
Injected into university registration pages. Uses MutationObserver to detect course/section DOM changes in real-time. Extracts professor names, course codes, section numbers (FR1, FR2).

Overlay Card UI
Rich professor profile card — overall rating, difficulty, grade charts, AI summary, strengths/weaknesses — rendered inline without leaving the registration page (FR4, FR5, DSR10).

Section Comparison View
Side-by-side comparison of multiple sections for the same course. Students identify best-fit by rating, grading, workload, or teaching style (FR7, DSR4).
ReactTailwind CSSRecharts

Schedule Builder
Multi-course semester planner evaluates full schedule against workload balance, grade difficulty, and professor quality. Returns ranked combinations (FR9, DSR7).
ReactTypeScriptTailwind CSS

Preferences & Profile
Students set workload, teaching style, grading strictness, and attendance priorities. Drives personalised match scores and recommendations shown in overlays (FR8, DSR6).
ReactLocal StateTailwind CSS
✍️
Review Submission
Students submit ratings and reviews via extension UI. Community upvote system surfaces most helpful reviews; filter by major, year, or cohort (FR10, DSR3).

Layer 2 — API Gateway & Auth Node.js · Express · OAuth 2.0 · TLS 1.3 · Redis
🚪
REST API Gateway
Single entry point for all extension requests. Routes to internal services, enforces rate limiting, request validation, and CORS policy (NFR1, NFR4).
Node.jsExpressTypeScript
🔐
Authentication & Session
OAuth 2.0 for user login. JWT-based session tokens. All traffic over TLS 1.3. Minimal permissions principle for extension (NFR5, NFR6, DSR9).
OAuth 2.0JWTTLS 1.3
⚡
Response Cache Layer
Redis-backed cache for hot professor profiles and rating aggregates. Ensures sub-2s load time at 95th percentile. Cache invalidation on new reviews (NFR1, NFR4).
RedisElastiCacheTTL
🌐
WebSocket / Push Gateway
Real-time notification delivery for professor follow alerts. Interfaces with Firebase FCM for cross-device push to the Chrome extension (FR11, NFR10).
WebSocketNode.jsFirebase FCM
