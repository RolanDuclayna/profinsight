# ProfInsight

ProfInsight is a full-stack web application that helps students evaluate
professors, compare sections, and plan schedules using real university data.

## Tech Stack
- Next.js + React
- Node.js + Express
- PostgreSQL + Prisma
- JWT + Magic-Link Authentication

## Local Setup

```bash
git clone https://github.com/yourname/profinsight
cd profinsight
cp .env.example .env
docker-compose up
npm install
npm run dev:web
npm run dev:api
```

Open http://localhost:3000
